## Algorithm

阅读并点评至少一篇英文技术文章。主要是为了学习英文，如果你英文不行，你基本上无缘技术高手，推荐http://Medium.com。

## Tip

学习至少一个技术技巧。主要是为了总结和归纳你在日常工作中遇到的知识点。
